## Section 2: Technical Details/Engine/Suspension

The 2028 Jeep Scrambler is built on a sturdy frame that's designed to withstand the rigors of off-road driving. The vehicle features a powerful 3.6L V6 engine that produces 285 horsepower and 260 lb-ft of torque. This engine is paired with an 8-speed automatic transmission that provides smooth, seamless shifting. The Scrambler also features a advanced suspension system that includes a five-link coil suspension in the front and a five-link leaf spring suspension in the rear. This system provides excellent articulation and stability, making it perfect for navigating rocky, uneven terrain.

| **Specification** | **Description** |
| --- | --- |
| Engine | 3.6L V6 |
| Horsepower | 285 |
| Torque | 260 lb-ft |

| Suspension | Five-link coil (front), five-link leaf spring (rear) |
| Ground Clearance | 10.8 inches |
| Approach Angle | 35.5 degrees |

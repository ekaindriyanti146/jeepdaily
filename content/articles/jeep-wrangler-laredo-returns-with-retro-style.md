## Section 2: Technical Details/Engine/Suspension

Under the hood of the Jeep Wrangler Laredo lies a potent 3.6-liter V6 engine, capable of producing 285 horsepower and 260 lb-ft of torque.

This powerplant is paired with a smooth-shifting eight-speed automatic transmission, allowing for seamless transitions between gears.

The Laredo's suspension system is equally impressive, featuring a robust five-link setup in the front and a trailing arm design in the rear.

This configuration enables the vehicle to tackle challenging off-road terrain with ease, while also providing a comfortable ride on paved surfaces.

The following table highlights the key specifications of the Jeep Wrangler Laredo:
| Feature | Specification |
| --- | --- |
| Engine | 3.6L V6 |
| Horsepower | 285 HP |
| Torque | 260 lb-ft |
| Transmission | 8-speed automatic |
| Suspension (Front) | Five-link |
| Suspension (Rear) | Trailing arm |
| Ground Clearance | 10.8 inches |
| Approach Angle | 33.5 degrees |

| Curb Weight | 4,175 lbs |

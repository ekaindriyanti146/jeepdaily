## Section 2: Technical Details/Engine/Suspension

The Jeep Gladiator Rubicon 392 is powered by a 6.4-liter V8 engine, producing 470 horsepower and 470 lb-ft of torque.

This powerful engine is paired with an eight-speed automatic transmission, providing smooth and responsive shifting.

The Rubicon 392 also features a robust suspension system, including a disconnecting front sway bar, heavy-duty shocks, and a performance-tuned exhaust system.

The vehicle's approach and departure angles have been improved, allowing for better off-road capability.

The Gladiator Rubicon 392 also features a range of advanced technologies, including a forward-facing camera, electronic locking front and rear axles, and a Rock-Trac 4x4 system.

The following table provides a detailed overview of the Gladiator Rubicon 392's technical specifications:

| Feature | Specification |
| --- | --- |
| Engine | 6.4-liter V8 |
| Horsepower | 470 hp |
| Torque | 470 lb-ft |
| Transmission | 8-speed automatic |
| Suspension | Disconnecting front sway bar, heavy-duty shocks |
| Approach Angle | 33.6 degrees |

| Ground Clearance | 11.1 inches |
| Curb Weight | 5,672 pounds |

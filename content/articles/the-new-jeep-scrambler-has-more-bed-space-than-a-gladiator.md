## Technical Details/Engine/Suspension

The new Jeep Scrambler is powered by a 3.6L V6 engine, producing 285 horsepower and 260 lb-ft of torque.

This robust engine is paired with a 6-speed manual transmission, allowing for seamless shifting and optimal control.

The Scrambler's suspension system features a five-link coil setup in the front and a five-link leaf spring setup in the rear, providing a smooth ride and exceptional stability.

The following table highlights the Scrambler's key technical specifications:
| Feature | Specification |
| --- | --- |
| Engine | 3.6L V6 |
| Horsepower | 285 |
| Torque | 260 lb-ft |
| Transmission | 6-speed manual |
| Suspension | Five-link coil (front), five-link leaf spring (rear) |
| Bed Space | 6.5 ft (78.7 in) |
| Towing Capacity | Up to 7,650 lbs |
| Ground Clearance | 11.1 in |
The Scrambler's impressive bed space, measuring 6.5 ft (78.7 in), surpasses that of the Gladiator, making it an ideal choice for those who need to haul heavy loads or equipment.

## Real World Performance

In real-world testing, the Jeep Scrambler has proven to be a force to be reckoned with. Whether you're tackling the toughest trails or simply need a reliable work vehicle, the Scrambler is up to the task.

## Section 2: Technical Details/Engine/Suspension

The Lexus LX 2026 500d Overtrail is powered by a 3.0-liter twin-turbo diesel engine, producing 304 horsepower and 516 lb-ft of torque.

This potent powerplant is paired with a 10-speed automatic transmission, which provides smooth and seamless shifting.

The vehicle's suspension system features a double-wishbone setup in the front and a four-link design in the rear, allowing for excellent articulation and stability.

The 500d Overtrail also boasts an impressive array of off-road technologies, including a multi-mode traction control system, crawl control, and a locking center differential.

The following table highlights the key technical specifications of the Lexus LX 2026 500d Overtrail:

| Category | Specification |
| --- | --- |
| Engine | 3.0-liter twin-turbo diesel |
| Horsepower | 304 hp |
| Torque | 516 lb-ft |
| Transmission | 10-speed automatic |
| Suspension | Double-wishbone (front), four-link (rear) |
| Traction Control | Multi-mode with crawl control and locking center differential |
| Ground Clearance | 8.9 inches |
| Approach/Departure Angles | 25/20 degrees |
| Wading Depth | 27.6 inches |
